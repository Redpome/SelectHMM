# SelectHMM
Select large-scale sequences of candidates from HMMER results

To use the SelectHMM, you should pre-install the HMMsearch, and install SelectHMM.pl as well as ExtractSequence.pl in the same dir.

If you use the data obtained by SelectHMM, please cite in your paper:
"Zhang TK, Liu CY, Zhang HY, Yuan ZH: An integrated approach to identify Cytochrome P450 superfamilies in plant species within the Malvids. In: Proceedings of 2017 5th international conference on bioinformatics and computational biology. Hong Kong: ACM; 2017: 11-16."
